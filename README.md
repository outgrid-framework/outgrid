# Outgrid

A CSS framework built for even faster development. Weild the power of CSS Grid in your next project.

---

## Install

### CDN Link

`https://cdn.jsdelivr.net/npm/outgrid-beta@0.1.5/outgrid.css`

### NPM

`npm -i outgrid-beta`

#### Import in your HTML file using :

`<link rel="stylesheet" href="./node_modules/outgrid-beta/outgrid.css">`
